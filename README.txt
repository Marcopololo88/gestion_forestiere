Plugin QGIS : Gestion Forestière
================================

Ce plugin permet la gestion des données forestières dans QGIS :
- consultation des parcelles et des surfaces et des essences d'arbres.
- saisie des travaux, traitements et prévisions.
- gestion des voisins et de la possession.
- Analyse des données
- export vers Excel

Auteur : Marc G.
Version : 1.0

Ce plugin est compatible avec QGIS 3.34+.
